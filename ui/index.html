BEARS Propulsion

Hybrid Rocket Analysis

Evaluate NASA CEA nozzle and mixture sweeps for an N2O and paraffin engine with ABS structure, then inspect interactive plots, raw CEA result points, and a preliminary 0D blowdown model seeded from the best CEA case.

Solver NASA CEA
Engine N2O / Paraffin + ABS
Units SI Internal

Analysis Setup

Set the CEA chamber condition and sweep ranges, then configure the preliminary 0D blowdown model that can run automatically from the highest-Isp CEA case.

Chamber Condition

Target chamber pressure goes into CEA; thrust and exit diameter are used only for post-CEA nozzle sizing.

Ae/At Sweep

Defaults to start 1, step 1, and an upper limit determined by the selected simulation cap.

Advanced options

Simulation Cap

Limit the sweep by calculated exit diameter or by an explicit maximum area ratio.

O/F Sweep

Oxidizer-to-fuel mass-ratio range scanned at each area ratio.

Reactant Temperatures

Fuel and oxidizer temperatures passed separately to CEA.

Preliminary 0D Blowdown Model

Advanced and manual setup for the preliminary 0D blowdown model. These controls stay under the shared advanced dropdown and remain separate from the CEA sections.

Advanced / Manual Overrides

These controls are hidden in basic mode. Enable an override to stop auto-deriving that field.

Advanced Policy And Empirical Inputs

These fields stay out of basic mode. In advanced mode they replace the project defaults and presets used by the basic first-pass sizing flow.

Regression Preset

The basic-mode regression preset selector remains the source of truth. Use Custom there if you want the advanced `a` and `n` values below to take effect.

Injector Policy

The basic-mode injector pressure-drop policy selector remains the source of truth. Advanced mode replaces it with the detailed delta-p controls below.

When the regression preset is Custom, the advanced `a` and `n` inputs are used. Otherwise the selected project preset supplies those values.

Advanced Geometry Overrides

These fields override the first-pass tank, injector, and grain geometry estimates directly.

Tank State Basis

In both modes, tank temperature sets the saturated nitrous pressure and liquid density. Advanced mode only changes whether tank mass and volume are manually overridden.

Feed

Equivalent lumped feed losses between tank and injector.

Simulation

Advanced time-marching and convergence settings for the transient solution.

Preliminary 0D Blowdown Model

High-level blowdown inputs live directly in the main form. Advanced/manual blowdown setup is under the shared advanced dropdown at the bottom.

CEA Seed

Uses the latest completed highest-Isp CEA case for Isp, O/F, c*, and ABS fraction. Tank state follows the main oxidizer temperature input.

Hidden Basic Defaults

Usable oxidizer fraction, injector Cd, injector hole count, port count, and fuel usable fraction use project defaults in basic mode unless you switch to advanced mode.

Live Derived Results

Read-only first-pass estimates derived from burn time + O/F + Isp + oxidizer temperature + fill policy, using the latest highest-Isp CEA case for the remaining seed values.
